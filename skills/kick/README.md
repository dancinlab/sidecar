# kick

`/kick <seed>` — wrap `hexa kick --seed "<seed>"` (the hexa-lang gap-breakthrough / discovery engine, aliased to `hexa drill`).

## Trigger

- Slash: `/kick <natural-language seed>` — all args join into `--seed`
- Natural language: *"kick this"*, *"gap breakthrough on X"*, *"discover for X"*, *"발산"*, *"돌파해줘"*, *"이거 hexa kick 해"*

## Advanced flags

For `--rounds N`, `--engine mk9|mk10`, call `hexa kick` directly — the slash form is intentionally NL-only (`bin/kick.sh` does `exec hexa kick --seed "$*"`).
