# sample-cost-routing — cost-benchmark paper starter

Minimal mirror of hexa-codex's `economics-routing-savings/` structure. Formula → Method → Results → Limitations → Reproducibility, with the cost sum identity + savings percent baked in.

Copy via:

```
/paper sample sample-cost-routing <your-slug>
```

Then replace the manifest, headline numbers, and reproducibility pointer. Persist raw `claude --bare -p --output-format json` stdout under `.verdicts/<slug>/` so every dollar figure is bit-for-bit recomputable.
