description: Main-flow return stack — `/trail` tracks where you were on the MAIN FLOW (the active goal) when you deviate into ANY side-task, so you remember to climb back. Deviations are NOT only cross-repo — a same-repo tangent or sub-fix off the active milestone strands the main flow just as easily; this covers both (an intra-repo tangent/sub-fix AND a cross-repo/upstream fix). Store is HOME-global (`~/.sidecar/trail.tape`): one stack for the current dive, so it covers intra-repo deviations AND survives `cd` when a detour does cross repos (anima → hexa-lang → codegen). Verbs — `push <return-target>` records a frame (timestamp + cwd repo + the text you want to return to) · `pop` removes the top frame and prints what to resume · bare/`show` renders the ladder (deepest = ★ NOW, root at the bottom) · `clear` wipes · `help`. LLM AUTO-USE (commons g74) — the calling LLM should run this WITHOUT being told: the instant work pivots from the active goal into ANY side-task — a same-repo tangent/sub-fix OR a cross-repo/upstream fix (e.g. "let me just fix this adjacent thing first" / "this needs a hexa-lang fix first") — run `/trail push <one-line return target = the parent milestone to resume>` BEFORE diving; the instant that sub-fix closes (PR merged / gap filed), run `/trail pop` and resume the frame it names. On a long yak-shave, `/trail` (show) re-surfaces the climb-back path so the main flow is never stranded. Distinct from `/draft` (cwd-local throwaway notes) and `/domain` (in-repo milestone tracker) — `/trail` is the main-flow deviation STACK (LIFO push/pop · intra OR cross repo). Triggers — "/trail", "트레일", "복귀 스택", "이탈 기록", "메인 흐름 표시", "breadcrumb", "return stack", "어디까지 했더라", "원래 뭐 하던", "곁가지", "detour", "climb back", "본 흐름 복귀".
